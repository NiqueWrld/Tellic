# Bundled ADB binaries

This directory contains a copy of Google's Android Platform Tools `adb` binary
plus its required Windows DLLs. The contents are:

- `adb.exe`
- `AdbWinApi.dll`
- `AdbWinUsbApi.dll`

These files are generated by `scripts/fetch-adb.ps1` and are excluded from git
(see `.gitignore`). They are bundled into the packaged Electron app via the
`extraResource` field in `forge.config.ts`, which copies the directory to
`<app>/resources/adb/`.

At runtime, the main process resolves the adb path with `getAdbPath()` in
[`src/adb-path.ts`](../../src/adb-path.ts):

- Dev:    `<project>/resources/adb/adb.exe`
- Packaged: `<app>/resources/adb/adb.exe`
- Fallback: `adb` from `PATH` (only if the bundled file is missing)

To refresh the bundled platform tools:

```pwsh
pnpm fetch:adb -Force
```
